# E-commerce Dashboard (Power BI)

This project is an E-commerce Sales Dashboard created using Power BI.
Power BI E-commerce Dashboard analyzing sales, customers, and product performance using interactive visuals.

## 📊 Key Features:
- Total Sales, Orders, Customers, Quantity
- Sales by Category and Month
- Customer analysis (Gender, Age)
- Channel performance (Amazon, Flipkart, Myntra)
- City-wise sales insights

## 🛠 Tools Used:
- Power BI
- CSV Files (Customer, Orders, Product)
